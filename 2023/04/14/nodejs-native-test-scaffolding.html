      NodeJS v16.17 / v18 introduced a new module named node:test which includes basic test scaffolding! This makes it real easy to whip up a super-basic JS test file and iterate on it, without having to set up a framework like Mocha or Jest.

      Save this to a file named with a .mjs extension, e.g. test.mjs:

      import {strictEqual} from 'assert'
import {test} from 'node:test';

test('runs a thing', () => {
  strictEqual('foo', 'bar')
})
