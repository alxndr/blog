## kipisi nanpa 40

反者道之動。弱者道之用。天下萬物生於有、有生於無。

| Toki Pona | Chinese | English
|-:|:-:|:-
|  |  | Return is the motion of the Way.
|  |  | Softening is its function.
|  |  | All things in the cosmos arise from being.
|  |  | Being arises from non-being.
