大成若缺、其用不弊。大盈若冲、其用不穷。大直若屈、大巧若拙、大辩若訥。躁胜寒、静胜热。淸静为天下正。

|  |  | Great skill seems clumsy,
|  |  | Great eloquence seems stammering.
